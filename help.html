üìñ How to Use

Exporting Conversations

  1. Open any WhatsApp conversation
  2. Click the green floating AI button (bottom-right corner)
  3. Select "Export Conversation"
  4. The conversation will be downloaded as a text file

Generating AI Responses

  1. Make sure you've configured your Gemini API key
  2. Open any WhatsApp conversation
  3. Click the floating AI button
  4. Select "Generate AI Response"
  5. Wait for the AI to analyze the conversation and generate a response
  6. Choose to either copy the response or insert it directly into the chat

Message Detection

The extension automatically detects both incoming and outgoing messages using these CSS selectors:

Incoming messages: .message-in
Outgoing messages: .message-out
Message containers: [data-testid="msg-container"]

üîß Technical Details

Message Extraction

The extension uses multiple selectors to reliably extract messages:

  • .selectable-text - Main message text
  • [data-testid="conversation-text"] - Alternative text selector
  • [data-testid="msg-time"] - Message timestamps
  • [data-pre-plain-text] - Sender information for group chats

AI Integration

The extension uses Google's Gemini 1.5 Flash model with these parameters:

API Endpoint: /v1/models/gemini-1.5-flash:generateContent
Temperature: 0.7 (balanced creativity)
Max Output Tokens: 1024
Top K: 40
Top P: 0.95

üêõ Troubleshooting

Extension Not Working

  • Refresh the WhatsApp Web page
  • Check if the extension is enabled in chrome://extensions/
  • Make sure you're on web.whatsapp.com (not other WhatsApp versions)

AI Responses Not Generating

  • Verify your Gemini API key is correctly entered
  • Check your internet connection
  • Ensure you haven't exceeded API rate limits
  • Try again in a few moments

Messages Not Extracting

  • Make sure the conversation is fully loaded
  • Scroll up to load older messages if needed
  • Refresh the page and try again

üîí Privacy & Security

  • All message processing happens locally in your browser
  • API key is stored securely in Chrome's sync storage
  • Conversations are only sent to Gemini when you explicitly request AI responses
  • No data is stored on external servers (except during API calls to Google)
  • Extension only works on web.whatsapp.com for security
